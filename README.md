# ADHD Risk Prediction Web App

This is a browser-based research prototype that estimates the likelihood of ADHD
using demographic, academic, and lifestyle variables.

## Purpose
- Academic research and thesis work
- Educational demonstration of predictive modeling
- Not intended for clinical diagnosis

## Features
- ADHD risk probability estimation
- Risk categorization (Low / Moderate / High)
- Display of top contributing factors
- Based on logistic regression

## Disclaimer
This tool is for research and educational purposes only and must not be used
for diagnosis or clinical decision-making.
